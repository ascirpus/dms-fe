# CedarStack Admin Guide

This guide covers workspace administration tasks. You need **Admin** or **Owner** role to perform most of these actions.

## Contents

| Guide | Description |
|-------|-------------|
| [Tenant Setup](tenant-setup.md) | Initial workspace configuration and organization settings |
| [User Management](user-management.md) | Inviting users, managing roles, and handling team members |
| [Document Types](document-types.md) | Creating and configuring document type categories |
| [Billing](billing.md) | Plans, usage limits, and subscription management |
| [Integrations](integrations.md) | API access, MCP integration, and webhooks |

## Role Requirements

| Action | Required Role / Capability |
|--------|---------------------------|
| Invite users | Admin+ (Manage Users) |
| Manage parties | Admin+ (Manage Parties) |
| Create/delete projects | Admin+ (Manage Projects) |
| Upload/delete documents | Admin+ (Manage Documents) |
| Configure document types | Admin+ (Manage Settings) |
| Manage billing | Owner only (Manage Billing) |
| Workspace settings | Admin+ (Manage Settings) |
